# Notification Component (Vue.js)

Домашнее задание по курсу Vue.js 3 — компонент уведомления.
Сделано по аналогии с уроком "Build a Notification Vue Component". Компонент можно переиспользовать сколько угодно раз с разными пропсами.

## Что делает

- показывает уведомление с заголовком и текстом
- текст передаётся через слот
- есть 4 типа: info, warning, error, success (цвет меняется в зависимости от типа)
- можно закрыть уведомление, нажав на крестик

## Использовано

- Vue.js 3 (через CDN, без сборки)
- Semantic UI (для стилей, тоже через CDN)

## Как запустить

Просто открыть index.html в браузере, ничего устанавливать не надо.
