# Vue.js Notification Component

Учебный компонент уведомления на Vue.js 3 (Options API, глобальная регистрация компонента через `.component()`), созданный по аналогии с уроком "Build a Notification Vue Component".

## Возможности

- переиспользуемый компонент `app-notification`;
- содержимое уведомления передаётся через `<slot>`;
- пропс `type` задаёт вид уведомления (`info`, `warning`, `error`, `success`) — от него зависит цвет через Semantic UI;
- пропс `title` задаёт заголовок уведомления;
- у обоих пропсов есть значения по умолчанию, поэтому компонент можно использовать без указания атрибутов;
- закрытие уведомления по клику на иконку крестика.

## Технологии

- HTML5
- Vue.js 3 (CDN, Options API, глобальные компоненты)
- Semantic UI (CDN) — стили для блока message

## Как запустить

Открыть файл `index.html` в браузере — установка зависимостей не требуется.
